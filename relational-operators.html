Less Than Operator

<

  • if the operands are numbers, then a numeric comparison is performed.
  • Ifthe operands are strings, then the character codes of each character will be compared.
  • If only one operand is a number then the other is converted to a number and a comparison is made.
  • If an operand is an object, call valueOf() and use its result to make a comparison whilst sticking to the rules stated above, if valueOf() is not available, call toString() and use that value according the the rules above.
  • If the operand is a Boolean, then it is converted to a number and a comparison is made.
